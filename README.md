# rg_projekat

### Deadpool vs Wolverine

## Advanced techniques
- Cubemaps 
- HDR & Bloom

## Key Bindings
- `WASD` - movement
  
- `F1` - open ImGui
  
- `B` - Blinn-Phong 
  
- `H` - HDR 
  
- `SPACE` - bloom 
  
- `Q` - decrease exposure

- `E` - increase exposure
  
- `ESC` - interrupts program execution
